Voici les parties 3 et 4 du Livrable 4

Ce projet est composé de différents dossiers :

	Fichiers_NPr => contient les fichiers xml contenant les NPr de chacun des fichiers textes que vous nous avez donné, que mes partenaires de projet ont réalisé.

	Fichiers_BD_geoName => contient initialement (déja construits) les fichiers xml avec les informations recupérées sur le site de geoName.

	Fichiers_GPX => contient les fichiers GPX réalisés avec les transformations xslt.

	Fichiers_KML => contient les fichiers KML réalisés avec les transformations xslt. 

Le dossier racine contient quant à lui directement les 3 fichiers permettant les différentes trasnformations xslt avec la récuperation des informations sur geaName, la transformation en fichier GPX(peu utile ici car la visualisation du trajet n'est pas propre), ainsi que le fichier permettant la transformation en KML.

Les informations récupérées sont à mon avis trop nombreuses, il faudrait un module permettant de déterminer de quelle entité nommée le texte fait référence. En effet "Bellevue" peut faire référence à diverses geolocalisations.

La visualisation des fichiers GPX peut se faire sur google map à l'adresse suivante:

https://www.google.com/maps/d/

La visualisation des fichiers KML peut se faire via le site de google earth en téléchargeant Google Earth Pro à l'adresse suivante :

https://www.google.fr/intl/fr_ALL/earth/versions/

Les transformations xslt sont déja pré-préparées avec une ouverture du fichier newProject.xpr, les dossiers contiennent déja des fichiers pré-fabriqués, mais vous pourrez lancer à nouveau les traitements si vous le désirez.
